Notes App — это простое многостраничное веб-приложение для создания и хранения заметок. Приложение позволяет добавлять, просматривать, редактировать и удалять заметки. Все данные сохраняются в базе данных SQLite.

Функционал:
1. Просмотр заметок — список всех заметок на главной странице
2. Добавление заметки — форма с полями для заголовка и текста
3. Редактирование заметки — изменение существующей заметки
4. Удаление заметки — удаление с подтверждением
5. Автоматическая дата — дата создания заметки добавляется автоматически
6. Сортировка — заметки отображаются от новых к старым

Технологии:
1. Python 3
2. Flask (веб-фреймворк)
3. SQLite (база данных)
4. SQLAlchemy (ORM)
5. HTML5
6. CSS3
7. Jinja2 (шаблонизатор)
