Notes App — это многостраничное веб-приложение для создания и хранения заметок. Приложение позволяет пользователю создавать, просматривать, редактировать и удалять заметки, а также искать их по тексту и фильтровать по категориям. Все данные сохраняются в базе данных SQLite.

Проект разработан в рамках лабораторной работы по дисциплине "Веб-технологии". Приложение демонстрирует работу с серверной частью на Flask, базой данных SQLite, шаблонизатором Jinja2 и адаптивным интерфейсом на HTML/CSS.

Функционал:
1. Создание заметки — добавление заметки с заголовком, текстом и выбором категории.
2. Просмотр заметок — список всех заметок на главной странице, отсортированный от новых к старым.
3. Редактирование — изменение заголовка, текста и категории существующей заметки.
4. Удаление — удаление заметки с подтверждением действия.
5. Поиск — мгновенный поиск по заголовку и тексту заметок без перезагрузки страницы.
6. Фильтрация по категориям — отображение заметок выбранной категории: Работа, Личное, Учёба, Идеи, Другое.
7. Автоматическая дата — дата создания заметки добавляется автоматически.
8. Страница "О проекте" — информация о приложении, технологиях и авторе.
9. Адаптивный интерфейс — корректное отображение на компьютерах, планшетах и телефонах.

Для разработки приложения использовались следующие технологии:
1. Бэкенд написан на Python с использованием веб-фреймворка Flask. Flask отвечает за маршрутизацию, обработку HTTP-запросов и передачу данных между клиентом и сервером.
2. База данных — SQLite. Для работы с базой данных используется SQLAlchemy — объектно-реляционное отображение, позволяющее работать с таблицами как с классами Python.
3. Шаблоны создаются с помощью Jinja2 — встроенного в Flask шаблонизатора. Он позволяет динамически генерировать HTML-страницы, использовать условные операторы и циклы.
4. Фронтенд реализован на HTML5 и CSS3. Стилизация выполнена с использованием Flexbox для гибкого расположения элементов и медиа-запросов для адаптивности под разные устройства.
5. Поиск и фильтрация реализованы на чистом JavaScript без использования сторонних библиотек. Фильтрация происходит на стороне клиента, что обеспечивает мгновенный отклик без перезагрузки страницы.
6. Контроль версий осуществляется с помощью Git. Код хранится в репозитории на GitHub.

В корневой папке проекта находится файл app.py, содержащий всю серверную логику, маршруты и настройки приложения. Файл requirements.txt содержит список зависимостей Python, необходимых для работы.
Папка instance создаётся автоматически при первом запуске и содержит файл базы данных notes.db.
Папка static содержит файл style.css со всеми стилями приложения.
Папка templates содержит все HTML-шаблоны. Базовый шаблон base.html задаёт общую структуру всех страниц и содержит навигационное меню. Шаблон index.html отображает главную страницу со списком заметок, полем поиска и кнопками фильтрации. Шаблон add.html предназначен для добавления новой заметки. Шаблон edit.html используется для редактирования существующей заметки. Шаблон about.html содержит информацию о проекте.

Для установки приложения необходимо склонировать репозиторий с GitHub. После клонирования нужно перейти в папку проекта.
Далее требуется создать виртуальное окружение Python для изоляции зависимостей. Активация виртуального окружения выполняется командой source venv/bin/activate для Linux и Mac или venv\Scripts\activate для Windows.
Затем необходимо установить все зависимости из файла requirements.txt с помощью pip.
После установки зависимостей можно запустить приложение командой python app.py. Сервер запустится, и приложение станет доступно по адресу http://localhost:5000.
