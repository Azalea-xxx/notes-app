О проекте

Notes App

Веб-приложение для создания и хранения заметок. Позволяет добавлять, редактировать, удалять заметки, а также искать их по тексту и фильтровать по категориям.

Технологии

  • Бэкенд: Python 3, Flask
  • База данных: SQLite, SQLAlchemy
  • Фронтенд: HTML5, CSS3, JavaScript
  • Шаблоны: Jinja2

Возможности

  • Создание, просмотр, редактирование и удаление заметок
  • Выбор категории (Работа, Личное, Учёба, Идеи, Другое)
  • Мгновенный поиск по заголовку и тексту
  • Фильтрация заметок по категориям
  • Автоматическое добавление даты создания
  • Адаптивный дизайн для мобильных устройств
